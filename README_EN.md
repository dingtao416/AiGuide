## 📖 Introduction

**AiGuide** is an open-source artificial intelligence learning guide designed to help developers and AI enthusiasts systematically learn AI-related knowledge. From AI fundamentals to cutting-edge technologies, it covers popular fields such as Machine Learning, Deep Learning, Large Language Models (LLM), RAG, and AI Agents.

> 🎯 **Vision**: Make AI accessible to everyone, master the core technologies

## ✨ Features

- 📚 **Comprehensive** - From beginner to advanced, covering core AI knowledge
- 🔥 **Cutting-edge** - Continuously updated with LLM, RAG, Agent and other hot technologies
- 💡 **Easy to understand** - Complex concepts explained in simple language
- 🛠️ **Practice-oriented** - Tool guides and best practices included
- 🌐 **Open source & Free** - Community contributions and sharing are welcome

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/dingtao416/AiGuide.git

# Navigate to the directory
cd AiGuide

# Install dependencies (if running the documentation site locally)
pnpm install

# Start the local development server
pnpm docs:dev
```
