# AIGuide 项目介绍

## 🎯 项目愿景

**AIGuide** 旨在成为最全面、最易懂的中文 AI 知识分享平台，帮助开发者和学习者系统性地掌握人工智能领域的核心知识。

## 📚 内容覆盖

### 理论基础
- **AI 基础知识**：人工智能的定义、发展历史、核心概念
- **机器学习**：监督学习、无监督学习、强化学习
- **深度学习**：神经网络、CNN、RNN、Transformer

### 前沿技术
- **大语言模型 (LLM)**：GPT、Claude、Llama、通义千问等
- **RAG 技术**：检索增强生成的原理与实践
- **AI Agent**：智能代理的架构与应用

### 开发实践
- **开发框架**：LangChain、LlamaIndex、Dify 等
- **开发工具**：PyTorch、HuggingFace、Ollama 等

## 🌟 项目特色

| 特点 | 描述 |
|------|------|
| 📖 系统全面 | 从基础到前沿，构建完整知识体系 |
| 🔄 持续更新 | 紧跟 AI 发展，及时更新内容 |
| 💡 通俗易懂 | 用简洁清晰的语言解释复杂概念 |
| 🛠️ 注重实践 | 理论结合代码，便于动手实践 |
| 🤝 开源共建 | 欢迎社区贡献，共同完善内容 |
