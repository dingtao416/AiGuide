## 🎯 阶段一：AI 基础（1-2 周）

### 学习目标
- 理解人工智能的基本概念和发展历史
- 掌握 AI 领域的核心术语
- 了解 AI 的主要应用场景

### 推荐内容
1. [什么是人工智能](/ai-basics/what-is-ai)
2. [AI 发展简史](/ai-basics/ai-history)
3. [AI 核心术语表](/ai-basics/ai-glossary)

### 补充学习
- 高中数学：线性代数、概率统计基础
- Python 编程基础

---

## 📊 阶段二：机器学习（2-4 周）

### 学习目标
- 理解机器学习的基本原理
- 掌握监督学习、无监督学习的区别
- 能够实现简单的 ML 算法

### 推荐内容
1. [机器学习入门](/machine-learning/introduction)
2. [监督学习](/machine-learning/supervised-learning)
3. [无监督学习](/machine-learning/unsupervised-learning)
4. [模型评估](/machine-learning/model-evaluation)

### 实践项目
- 使用 scikit-learn 完成分类任务
- 实现线性回归预测房价

---

## 🧠 阶段三：深度学习（3-6 周）

### 学习目标
- 理解神经网络的工作原理
- 掌握 CNN、RNN 的结构和应用
- 深入理解 Transformer 架构

### 推荐内容
1. [神经网络基础](/deep-learning/neural-networks)
2. [卷积神经网络 (CNN)](/deep-learning/cnn)
3. [循环神经网络 (RNN)](/deep-learning/rnn)
4. [Transformer 架构](/deep-learning/transformer)

### 实践项目
- 使用 PyTorch 实现 MNIST 手写数字识别
- 训练一个简单的文本分类模型

---

## 🚀 阶段四：前沿应用（4-8 周）

### 学习目标
- 理解大语言模型的原理和应用
- 掌握 RAG 技术的实现方法
- 了解 AI Agent 的架构设计

### 推荐内容
1. [大语言模型](/llm/what-is-llm)
2. [Prompt Engineering](/llm/prompt-engineering)
3. [RAG 技术](/rag/what-is-rag)
4. [AI Agent](/agent/what-is-agent)

### 实践项目
- 使用 LangChain 构建问答系统
- 实现基于 RAG 的知识库检索
- 开发一个简单的 AI Agent

## 💡 学习建议

### ✅ 推荐做法
- 理论与实践结合，边学边做
- 阅读经典论文，了解技术原理
- 参与开源项目，积累实战经验
- 加入学习社区，与他人交流

### ⚠️ 避免误区
- 不要只看教程不动手
- 不要贪多嚼不烂，循序渐进
- 不要忽视基础数学知识
- 不要闭门造车，多交流学习
