# 什么是人工智能

## 定义

**人工智能（Artificial Intelligence，AI）** 是计算机科学的一个分支，致力于创建能够执行通常需要人类智能的任务的系统。这些任务包括学习、推理、问题解决、感知、语言理解等。

> "人工智能是让机器表现出人类智能行为的科学与工程" — John McCarthy（AI 之父）

## AI 的主要类型

### 按能力分类

| 类型 | 描述 | 示例 | 现状 |
|------|------|------|------|
| **弱 AI / 窄 AI** | 专注于单一任务 | Siri、AlphaGo、ChatGPT | ✅ 已实现 |
| **强 AI / 通用 AI** | 具有人类级别的通用智能 | 科幻电影中的 AI | ⏳ 研究中 |
| **超级 AI** | 超越人类的智能 | 尚无 | 🔮 理论阶段 |

### 按方法分类

```
人工智能
├── 符号 AI（传统 AI）
│   ├── 专家系统
│   ├── 知识图谱
│   └── 规则引擎
│
└── 机器学习
    ├── 传统机器学习
    │   ├── 监督学习
    │   ├── 无监督学习
    │   └── 强化学习
    │
    └── 深度学习
        ├── CNN（计算机视觉）
        ├── RNN/Transformer（NLP）
        └── 生成式 AI（AIGC）
```

## AI 的核心能力

### 1. 感知（Perception）
- 计算机视觉：理解图像和视频
- 语音识别：理解人类语音
- 自然语言处理：理解文本

### 2. 推理（Reasoning）
- 逻辑推理
- 知识推理
- 常识推理

### 3. 学习（Learning）
- 从数据中学习模式
- 从反馈中改进
- 迁移学习

### 4. 行动（Action）
- 决策制定
- 规划执行
- 与环境交互

## AI 发展简史

```
1950s - 1960s    AI 诞生期
├── 1950: 图灵测试提出
├── 1956: 达特茅斯会议，"人工智能"一词诞生
└── 1966: ELIZA 聊天程序

1970s - 1980s    专家系统时代
├── 知识工程兴起
├── 专家系统商业化
└── 第一次 AI 寒冬

1990s - 2000s    机器学习崛起
├── 统计学习方法
├── 支持向量机
└── 深蓝击败国际象棋冠军

2010s            深度学习革命
├── 2012: AlexNet 图像识别突破
├── 2016: AlphaGo 击败围棋世界冠军
└── 2017: Transformer 架构提出

2020s            大模型时代
├── 2020: GPT-3 展现惊人能力
├── 2022: ChatGPT 引爆全球
└── 2023+: 多模态、Agent、具身智能
```

## AI 与相关概念

```
┌─────────────────────────────────────────────────┐
│              人工智能 (AI)                       │
│    ┌─────────────────────────────────────┐     │
│    │         机器学习 (ML)                 │     │
│    │    ┌─────────────────────────┐      │     │
│    │    │     深度学习 (DL)         │      │     │
│    │    │  ┌───────────────────┐  │      │     │
│    │    │  │  生成式 AI (GenAI) │  │      │     │
│    │    │  └───────────────────┘  │      │     │
│    │    └─────────────────────────┘      │     │
│    └─────────────────────────────────────┘     │
└─────────────────────────────────────────────────┘
```

- **AI**：最广泛的概念，包含所有让机器智能化的技术
- **机器学习**：AI 的子集，让机器从数据中学习
- **深度学习**：ML 的子集，使用深度神经网络
- **生成式 AI**：能够生成新内容的 AI，如 GPT、Stable Diffusion

## 学习 AI 需要什么基础？

### 编程能力
- Python（必须）
- 基本的数据结构和算法

### 数学基础
- 线性代数（向量、矩阵运算）
- 概率统计（概率分布、贝叶斯）
- 微积分（导数、梯度）

### 机器学习基础
- 基本概念和术语
- 常见算法原理
- 模型评估方法

```
   ↓

```
