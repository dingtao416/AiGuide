# 计算机视觉基础

## 什么是计算机视觉

**计算机视觉（Computer Vision, CV）** 是让计算机理解和处理图像/视频的AI技术。

## 主要任务

| 任务 | 说明 | 应用 |
|------|------|------|
| 图像分类 | 识别图像类别 | 物体识别 |
| 目标检测 | 定位+分类 | 自动驾驶 |
| 图像分割 | 像素级分类 | 医学影像 |
| 人脸识别 | 身份验证 | 安防 |
| 姿态估计 | 人体关键点 | 动作捕捉 |
| 图像生成 | 创建图像 | AI绘画 |

## 基础概念

### 图像表示
```
图像 = 高度 × 宽度 × 通道
RGB图像: H × W × 3
灰度图: H × W × 1
```

### 卷积操作
CNN 的核心，提取图像特征。

## 经典网络

- **AlexNet**：深度学习革命
- **VGG**：深层网络
- **ResNet**：残差连接
- **YOLO**：实时检测
- **ViT**：Vision Transformer

## 工具库

- **OpenCV**：传统视觉
- **torchvision**：PyTorch视觉
- **Detectron2**：目标检测
