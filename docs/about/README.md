# 关于 AI Guide

## 🎯 项目简介

**AI Guide** 是一个开源的人工智能知识分享平台，致力于为 AI 学习者提供系统、全面的学习资源。

### 我们的目标

- 📚 提供**系统化**的 AI 学习路径
- 💡 用**通俗易懂**的语言讲解复杂概念
- 🔧 分享**实用**的工具和框架教程
- 🌟 追踪**最新**的 AI 技术动态

## 📖 内容覆盖

```
AI Guide
├── AI 基础 - 人工智能入门概念
├── 机器学习 - 监督/无监督/强化学习
├── 深度学习 - 神经网络、CNN、RNN、Transformer
├── 大语言模型 - GPT、Claude、LLaMA、Qwen
├── RAG 技术 - 检索增强生成
├── AI Agent - 智能体开发
├── 开发框架 - LangChain、LlamaIndex
├── 计算机视觉 - 图像处理与生成
└── 自然语言处理 - 文本分析与生成
```

## 🤝 参与贡献

我们欢迎所有形式的贡献：
