# AI 学习书籍推荐

精选人工智能、机器学习、深度学习领域的经典书籍。

## 🎯 入门必读

| 书籍 | 作者 | 推荐理由 |
|------|------|----------|
| 《动手学深度学习》 | 李沐 等 | 理论+代码，开源免费，强烈推荐 |
| 《Python机器学习》 | Sebastian Raschka | 实战导向，适合入门 |
| 《机器学习》(西瓜书) | 周志华 | 国内经典教材，理论扎实 |
| 《深度学习入门》(鱼书) | �的圭吾 | 从零实现神经网络，易懂 |
| 《统计学习方法》 | 李航 | 机器学习理论经典 |

## 🧠 深度学习

| 书籍 | 作者 | 推荐理由 |
|------|------|----------|
| 《深度学习》(花书) | Goodfellow 等 | 深度学习圣经 |
| 《神经网络与深度学习》 | 邱锡鹏 | 国内优秀教材，开源 |
| 《动手学深度学习》 | 李沐 等 | PyTorch/TensorFlow 双版本 |
| 《深度学习进阶：自然语言处理》 | 斋藤康毅 | NLP 方向入门 |

## 🤖 大语言模型

| 书籍 | 作者 | 推荐理由 |
|------|------|----------|
| 《大规模语言模型》 | 张俊林 | 国内 LLM 系统性著作 |
| 《ChatGPT 原理与应用开发》 | 皮特 | 实战开发指南 |
| 《Prompt Engineering Guide》 | DAIR.AI | 提示工程权威指南（在线） |
| 《Build a Large Language Model》 | Sebastian Raschka | 从零构建 LLM |

## 📊 数学基础

| 书籍 | 作者 | 推荐理由 |
|------|------|----------|
| 《线性代数应该这样学》 | Sheldon Axler | 线代入门经典 |

## 💻 编程实战
